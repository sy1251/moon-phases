Phases of Moon
==============

This is a SVG project for *Drawing on Web* class with icons that show the **phases of moon**.

There are 5 icons included.

- Waxing crescent
- First quarter
- Full moon
- Last quarter
- Waning gibbous

[Click to see the project](http://i6.cims.nyu.edu/~sy1251/380/asg2_new/)
